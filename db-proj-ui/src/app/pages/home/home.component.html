Movie Insights
Welcome to Movie Insights! Movie Insights, provides historical analyses of movie production factors and their correlations with trends in movie performance. Various gauges of performance are based on monetary and/or ratings success (or significant lack thereof for inquiries towards which factors perform poorly). It is the developers' hope that these insights will provide meaningful guidance in the user's next movie activity - whether an investor looking for their next production, or a movie acifianados next great viewing. Enjoy!
The developers have built this application with well over 250,000 unique data points as the underlying database. Well-known sources IMDb.com and Kaggle.com are the basis of this data with the various tools implemented by the developers via complex SQL queries.
Historical performances are computed – based the aforementioned metrics – for actors, countries, genres, production companies. All of these results are plotted over a user's selected time range with the hopes that trends might be recognized to be used towards a more informed future movie choice.
